# リリース方針

リリースには、ソース配布物と wheel を含めます。

モデル重みとデータセットは、ソースコードとは別の配布物として扱います。配布する場合は、それぞれのライセンス、利用条件、安全上の注意を明記してください。

## GitHub Release

タグ `v*.*.*` を push すると、GitHub Actions の release workflow が Python パッケージをビルドし、GitHub Release に成果物を添付します。

## 含めないもの

- 成人向けサンプルメディア
- 権利不明の画像・動画・データセット
- 大きなモデル重み
- ユーザーの処理対象ファイル
