# モデルカードテンプレート

## モデル名

## 想定用途

成人・同意済み・権利処理済み素材に対するモザイク支援のみ。

## 禁止用途

- 未成年または若年に見える人物が含まれる素材
- 同意なく撮影・共有された素材
- 違法素材
- モザイク除去、復元、逆モザイク用途

## 学習データ

データの由来、ライセンス、合成データか実写データか、権利確認方法を記載します。

## 評価

Recall、False Negative count、Mask coverage、Temporal stability、Precision、処理速度を記載します。

## 制限事項

苦手な構図、ブラー、画面端、重なり、複数人、低照度など、既知の失敗条件を記載します。
